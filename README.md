# Python Banking Management System

A Python based banking management system developed as part of my MSc Computer Science coursework at Birmingham City University.

The project was designed to simulate core banking operations through both a command line system and a Tkinter graphical user interface.

## Features
- Admin login system
- Customer account management
- Deposits and withdrawals
- Fund transfers between accounts
- Management reporting
- Customer profile updates
- Transaction validation and error handling
- Tkinter GUI interface

## Technologies Used
- Python
- Tkinter
- Object Oriented Programming (OOP)

## Project Structure
- `bank_system.py` → Main banking logic
- `bank_GUI.py` → Tkinter graphical interface
- `customer_account.py` → Customer account operations
- `admin.py` → Admin functionality and permissions

## How to Run
1. Download or clone the repository
2. Open the project in VS Code or another Python IDE
3. Run:

```bash
python bank_GUI.py
```
